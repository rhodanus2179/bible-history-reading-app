# 聖書の歴史を旅する365日

聖書を歴史の流れに沿って365日で読み進める、ブラウザ完結型の通読支援アプリです。

## 主な機能

- 365日分の歴史順通読計画
- 読了状況と進捗率の保存
- 日ごとのメモ
- 通読表の検索・絞り込み
- 聖書史タイムライン
- 「ともに聴く聖書」の各章への直接リンク
- ライト／ダークモード
- JSONバックアップ・復元
- モバイル対応

## 使い方

`index.html` をブラウザで開くか、GitHub Pagesで公開して利用します。進捗やメモはブラウザの `localStorage` に保存されます。

## データについて

通読計画は、いのちのことば社 新改訳聖書センターの「聖書 新改訳 通読表【歴史順】」をもとに構成しています。

聖書本文・音声はアプリ内に収録せず、一般財団法人日本G&M文化財団の「ともに聴く聖書」へリンクします。

## 現在のバージョン

v1.2.1

## 構成

- `index.html` — アプリの入口
- `styles.css` — 表示スタイル
- `data/plan-*.js` — 365日分の通読計画
- `data/books.js` — 聖書66巻のPRSリンクコード
- `js/app-core.js` — 共通処理と聖書リンク解析
- `js/app-ui.js` — 画面描画、進捗、メモ、設定

## 注意

進捗やメモは端末・ブラウザごとに保存されます。ブラウザデータの削除や端末変更に備えて、アプリの設定画面からバックアップしてください。
