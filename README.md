# 聖書の歴史を旅する365日

聖書を歴史の流れに沿って365日で読み進める、単一HTMLの通読支援アプリです。

## 主な機能

- 365日分の歴史順通読計画
- 読了状況と進捗率の保存
- 日ごとのメモ
- 通読表の検索・絞り込み
- 聖書史タイムライン
- 「ともに聴く聖書」の各章への直接リンク
- ライト／ダークモード
- JSONバックアップ・復元
- モバイル対応

## 使い方

`index.html` をブラウザで開くだけで利用できます。進捗やメモはブラウザの `localStorage` に保存されます。

## データについて

通読計画は、いのちのことば社 新改訳聖書センターの「聖書 新改訳 通読表【歴史順】」をもとに構成しています。

聖書本文・音声はアプリ内に収録せず、一般財団法人日本G&M文化財団の「ともに聴く聖書」へリンクします。
